你每天有多少时间在"找东西"：点开启动台翻应用、进文件夹翻文件、切到浏览器算个数、翻聊天记录找刚才复制的链接。这些零碎操作加起来很可观。启动器（Launcher）就是来消灭这些零碎的：一个快捷键唤出输入框，敲几个字，要啥来啥。这一篇带你认识它。

把启动器想成你桌上的"万能对讲机"：喊一声应用名就打开、说个算式就出结果、说个文件名就跳过去——不用在桌面和文件夹之间来回切。

## 什么是启动器

启动器是一个全局快捷键唤出的输入框，它在后台索引了你的应用、文件、书签，并能执行各种"动作"：

- **Spotlight**：macOS 自带，免费，基础搜索够用。
- **Alfred**：macOS 老牌启动器，免费版够用，付费 Powerpack 解锁工作流。
- **Raycast**：macOS 新秀，免费且扩展生态强，界面现代，社区插件多。

Windows 上有类似的 `PowerToys Run`、`Listary`；Linux 有 `Albert`、`Ulauncher`。

## 高频用法

### 1. 打开应用（最常用）

不用去启动台翻，按快捷键（如 Raycast 的 `Option+Space`）敲 `chr` 就出 Chrome，回车打开。比鼠标找图标快得多。

### 2. 搜文件与文件夹

敲文件名片段，直接跳到文件或打开所在文件夹。找"上周的需求文档"不用翻一堆目录。

### 3. 即时计算与换算

直接输入 `128 * 7` 出结果；输入 `100 usd to cny` 出汇率换算；输入 `1h to s` 出 3600。不用开计算器或浏览器。

### 4. 剪贴板历史

Raycast / Alfred 能记住你复制过的多条内容，唤出后选一条粘贴。再也不怕"刚复制的链接被新复制覆盖了"（详见剪贴板管理那篇）。

### 5. 系统控制

一键锁屏、清空回收站、切换深色模式、连蓝牙设备——不用去系统设置里翻。

## Raycast 的扩展生态

Raycast 强在用"扩展（Extension）"：社区做了成千上万个，装了之后启动器能直接：

- 查 GitHub 通知、创建 Issue
- 控制 Spotify 播放
- 查快递、查天气
- 生成 UUID、时间戳
- 管理窗口布局

这些原本要开好几个网站或 App 的事，现在都在同一个输入框里完成。

## 怎么挑

- **macOS 想零成本上手**：先用自带 Spotlight，不够再试 Raycast（免费且强）。
- **想要最强可玩性**：Raycast 的扩展生态目前最活跃。
- **Windows / Linux**：用 PowerToys Run / Ulauncher 之类的对应工具，思路一致。

## 常见新手坑

- **装了却不用**：启动器价值在"天天用"，刚开始不习惯，逼自己一周就离不开了。
- **还用鼠标找应用**：明明一次快捷键搞定，惯性使然又去点启动台，得刻意练。
- **没开剪贴板历史**：复制粘贴来回丢内容，其实启动器已帮你存着。
- **忽略扩展**：只当搜索框用，浪费了 Raycast 能直连各种服务的能力。

## 更多实战：用 Raycast 一天省下的操作

举个真实场景：你正在写文档，要查一个 GitHub Issue、算一下 128 乘 7、再把刚复制的链接粘进表格。

不用启动器：切到浏览器开 GitHub、切回、开计算器、切回、发现链接被覆盖了、再去找……至少五六次上下文切换。

用 Raycast：按 `Option+Space`，输入 issue 回车看 GitHub；再 `Option+Space` 输入 `128*7` 出结果；剪贴板历史里翻出刚才的链接粘贴。全程键盘，上下文不丢。

## 常见认知误区

启动器这块，最大的误区是"装了却不用"。很多人装完 Raycast 还是习惯性去启动台翻图标，因为用鼠标是肌肉记忆，得刻意练一周才能形成新习惯——别半途而废。第二个误区是"只当搜索框用"——忽略了扩展生态，Raycast 能直连 GitHub、Spotify、查快递天气，这些原本要开好几个网站的事，现在一个输入框搞定。第三个误区是"Windows 和 Linux 没这东西"——其实有 PowerToys Run、Ulauncher 之类对应工具，思路完全一致，只是名字不同。第四个误区是"启动器会拖慢电脑"——它平时只是后台索引，占用极低，不用担心。第五个误区是"免费版不够用"——macOS 自带 Spotlight 和 Raycast 免费版已经覆盖绝大多数日常需求，不必为付费功能买单。启动器的本质是"用键盘代替鼠标找东西"，养成习惯后，你回不去那种点点点的慢节奏。
