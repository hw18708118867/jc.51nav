很多人用终端还是"敲完一行、按回车、再慢慢敲下一行"，遇到要改前面一个字母，得按住退格删到那、改完再重敲后面——慢得离谱。其实终端有一套快捷键，让光标移动、翻历史、清屏都瞬间完成。这一篇把最值钱的几个记牢，你就不想再用鼠标了。

把快捷键想成钢琴的指法：单个键平平无奇，组合起来你就能"弹快板"而不是"单指戳"。

## 光标移动

```text
Ctrl + A    跳到行首（Home）
Ctrl + E    跳到行尾（End）
Alt + B     光标左移一个词
Alt + F     光标右移一个词
Ctrl + 左右  也是按词移动（很多终端支持）
```

改一条长命令中间某个参数，用 `Ctrl+A/E` 配合左右键，比退格删到那快十倍。

## 删改

```text
Ctrl + U    删除光标到行首的所有内容
Ctrl + K    删除光标到行尾的所有内容
Ctrl + W    删除光标前的一个词
Ctrl + Y    粘贴刚才 Ctrl+U/K/W 删掉的内容（撤销删除）
```

`Ctrl+U` 一键清空整行重来；`Ctrl+W` 删词比一个个退格优雅。

## 命令历史

```text
↑ / ↓       上一条 / 下一条历史命令
Ctrl + R    反向搜索历史（输入关键词，自动匹配用过的命令）
!!          重复上一条命令（常配合 sudo：sudo !!）
!abc        执行最近一条以 abc 开头的命令
```

`Ctrl+R` 是宝藏：你上周跑过一条很长的部署命令，不用翻记录，按 `Ctrl+R` 输入关键词就能召回。

## 进程控制

```text
Ctrl + C    中止当前正在运行的程序
Ctrl + Z    把当前程序挂到后台暂停（用 fg 唤回）
Ctrl + L    清屏（等同于 clear 命令）
```

程序卡死或跑飞，`Ctrl+C` 是首选终止方式。

## 复制粘贴

```text
Ctrl + Shift + C    复制（终端里 Ctrl+C 是被占用的，复制得加 Shift）
Ctrl + Shift + V    粘贴
```

注意：终端里 `Ctrl+C` 是"中止"不是"复制"，这是新手最常踩的坑。复制粘贴要加 `Shift`。

## 标签页与分屏

如果用的是带标签的终端（如 macOS 终端、iTerm2、VS Code 集成终端）：

```text
Ctrl + Shift + T    新建标签页
Ctrl + Shift + ]    切换到下一个标签页
```

多开标签能一边跑服务、一边敲命令，互不干扰。

## 常见新手坑

- **复制用 Ctrl+C**：在终端里这会中止程序，复制得用 `Ctrl+Shift+C`。
- **改长命令用退格删到头**：应该用 `Ctrl+A` 跳行首、`Ctrl+W` 删词。
- **重复长命令手敲**：用 `Ctrl+R` 搜历史召回，或 `!!` 重跑上一条。
- **忘了 `Ctrl+L` 清屏**：屏幕乱了还用 `clear` 敲字，其实一个组合键搞定。

## 小测验

- 问题1：终端里想复制文字，正确组合？答案：`Ctrl+Shift+C`（Ctrl+C 是中止）。
- 问题2：怎么快速召回上周用过的某条长命令？答案：按 `Ctrl+R` 输入关键词反向搜索历史。
- 问题3：光标跳到行首、删到行尾分别用什么？答案：`Ctrl+A` 跳行首，`Ctrl+K` 删到行尾。

## 更多实战：改一条长命令中间的参数

假设你敲了一条长命令，跑完发现中间一个路径写错了：

```bash
rsync -avz /var/www/old-site/ user@server:/var/www/new-site/
```

用快捷键改：

1. `Ctrl+A` 跳到行首，再 `Alt+F` 按词右移，到 `old-site` 那词。
2. `Ctrl+W` 删掉这个词，输入正确的 `prod-site`。
3. `Ctrl+E` 跳回行尾，`Enter` 重跑。

全程没用一次退格删到头，比手改快太多。

## 召回上周的长命令

上周你跑过一条复杂的部署命令，忘了具体写法。按 `Ctrl+R`，输入 `deploy`，终端自动召回最近一条含 deploy 的命令，回车即重跑，或左右键编辑后再跑。

## 常见认知误区

快捷键这块，最多人栽在"复制"上。第一，在终端里用 `Ctrl + C` 复制——这是中止程序，复制得加 `Shift` 用 `Ctrl + Shift + C`，这个坑几乎人人踩过一次。第二，改长命令用退格删到头——应该用 `Ctrl + A` 跳行首、`Ctrl + K` 删到行尾、`Ctrl + W` 删词，快得多。第三，重复长命令手敲——用 `Ctrl + R` 反向搜索历史召回，或 `!!` 重跑上一条。第四，在文件树里翻找——`Ctrl/Cmd + P` 是编辑器、`Ctrl + R` 是终端历史，但找文件用编辑器、找命令用终端历史，别混。第五，忘了 `Ctrl + L` 清屏——屏幕乱了还手敲 `clear`，其实一个组合键搞定。第六，觉得快捷键太多记不住——其实核心就那几个：移动用 `Ctrl + A/E`、删改用 `Ctrl + U/K/W`、历史用 `↑/Ctrl+R`、清屏 `Ctrl+L`，练熟这八个就够日常。

## 这一篇你该记住的

- 移动：`Ctrl+A` 行首、`Ctrl+E` 行尾、`Alt+左右` 按词跳。
- 删改：`Ctrl+U` 删到行首、`Ctrl+K` 删到行尾、`Ctrl+W` 删词、`Ctrl+Y` 粘贴回来。
- 历史：`↑↓` 翻、`Ctrl+R` 搜、`!!` 重跑上一条。
- 控制：`Ctrl+C` 中止、`Ctrl+L` 清屏。
- 复制：`Ctrl+Shift+C/V`，别用 `Ctrl+C`。
